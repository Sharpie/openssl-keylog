OpenSSL::Keylog
===============

An extension to the Ruby OpenSSL library that logs session keys in [NSS Key Log Format][nss-format].

  [nss-format]: https://developer.mozilla.org/en-US/docs/Mozilla/Projects/NSS/Key_Log_Format
